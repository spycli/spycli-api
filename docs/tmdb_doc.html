TMDB API Documentation

This document provides an overview of the endpoints available for the TMDB API integration, allowing for searches and detailed season fetches.

Endpoints

/tmdb/search

Performs a search across multiple types of TMDB media, including movies, TV shows, and people, based on a query string.

Method: GET

Required query parameter: query - The search string.

Example request: GET /tmdb/search?query=Inception

/tmdb/fetch

Fetches the structure of seasons and episodes for a specified TV series by its TMDB ID.

Method: GET

Required query parameter: id - The TMDB ID of the TV series.

Example request: GET /tmdb/fetch?id=1399 (Where "1399" is the TMDB ID for "Game of Thrones")
