MangaClient API Documentation

This document outlines the usage and examples for the MangaClient API endpoints.

Routes

/manga/search

Searches for manga based on a query string. Returns a list of manga that match the query.

Required query parameters: source (string, available sources: mangasee123, mangareader, mangapill, mangapark, mangahere, mangadex), query (string).

GET /manga/search?source=mangadex&query=One%20Piece

/manga/info

Fetches detailed information for a specific manga identified by its ID.

Required query parameters: source (string, available sources: mangasee123, mangareader, mangapill, mangapark, mangahere, mangadex), id (string).

GET /manga/info?source=mangadex&id=259dfd8a-f06a-4825-8fa6-a2dcd7274230

/manga/chapters

Retrieves the pages of a specific manga chapter, based on the chapter ID.

Required query parameters: source (string, available sources: mangasee123, mangareader, mangapill, mangapark, mangahere, mangadex), chapterId (string).

GET /manga/chapters?source=mangadex&chapterId=5f7891b4-f048-4516-9c75-7bcd6dbd1451
