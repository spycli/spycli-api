MoviesDrive API Documentation

This document provides usage examples for the MoviesDrive API routes.

Routes

/moviesdrive/trending

Fetches a list of trending movies. Optional query parameter: page (integer).

GET /moviesdrive/trending?page=2

/moviesdrive/search

Searches for movies based on a query string. Required query parameter: query (string).

GET /moviesdrive/search?query=breaking+bad

/moviesdrive/detail

Fetches details for a specific movie. Required query parameter: id (string).

GET /moviesdrive/detail?id=breaking-bad-season-1-5

/moviesdrive/quality

Fetches quality and content links for a movie. Required query parameter: url (string).

GET /moviesdrive/quality?id=13820

/moviesdrive/play

Streams a movie from a provided URL. Required query parameter: url (string).

GET /moviesdrive/play?id=2ejavosoirb4ihe
