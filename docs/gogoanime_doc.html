GogoAnime API Documentation

This document provides usage examples for the GogoAnime API routes.

Routes

/gogoanime/trending

Fetches a list of trending anime. No query parameters required.

GET /gogoanime/trending

/gogoanime/search

Searches for anime based on a query string. Required query parameter: query (string).

GET /gogoanime/search?query=One+Piece

/gogoanime/detail

Fetches details for a specific anime. Required query parameter: id (string).

GET /gogoanime/detail?id=one-piece

/gogoanime/episode

Fetches streaming URLs for a specific anime episode. Required query parameter: id (string).

GET /gogoanime/episode?id=one-piece-episode-1

/gogoanime/episode/download

Fetches the download URL for a specific anime episode. Required query parameter: id (string).

GET /gogoanime/episode/download?id=one-piece-episode-1
