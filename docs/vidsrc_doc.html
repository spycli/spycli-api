VIDSrc API Documentation

This document provides usage examples for the VIDSrc API routes.

Routes

/vsc/vidsrc/movie

Fetches video source information for a specific movie. Required query parameter: IMDB or TMDB ID (string).

GET /vsc/vidsrc/movie?id=tt1477834

/vsc/vsrcme/movie

Fetches video source information for a specific movie from VSrcMe. Required query parameter: IMDB or TMDB ID (string).

GET /vsc/vsrcme/movie?id=297802

/vsc/vidsrc/tv

Fetches video source information for a specific TV show episode. Required query parameters: IMDB or TMDB ID (string), season (string), episode (string).

GET /vsc/vidsrc/tv?id=66732&season=1&episode=2

/vsc/vsrcme/tv

Fetches video source information for a specific TV show episode from VSrcMe. Required query parameters: IMDB or TMDB ID (string), season (string), episode (string).

GET /vsc/vsrcme/tv?id=tt4574334&season=1&episode=2
