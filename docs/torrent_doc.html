Torrent API Documentation

This document provides usage examples for the Torrent API routes, including how to search across various torrent sites.

Routes

/torrent/search/all

Performs a search across all configured torrent sites. Optional parameters: query (string), limit (integer).

GET /torrent/search/all?query=ubuntu&limit=10

/torrent/search/site

Performs a search on a specific torrent site. Required parameters: query (string), site (string). Optional parameter: limit (integer).

GET /torrent/search/site?query=ubuntu&site=ThePirateBay&limit=10
