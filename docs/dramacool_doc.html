DramaCool API Documentation

This document outlines the usage and examples for the DramaCool API endpoints.

Routes

/dramacool/search

Searches for dramas based on a query string. Returns a list of dramas that match the query.

Required query parameter: query (string).

GET /dramacool/search?query=The%20Impossible%20Heir

/dramacool/info

Fetches detailed information for a specific drama identified by its ID.

Required query parameter: id (string).

Optional page parameters: page (int, deafault page 1).

GET /dramacool/info?id=drama-detail/royal-loader

/dramacool/streaming

Retrieves streaming links for a drama episode, based on the episode ID, media ID, and optional server preference.

Required query parameters: episodeId (string), mediaId (string).

Optional query parameters: server (string, available servers: asianload, mixdrop, streamtape, streamsb, mangahere, mangadex).

GET /dramacool/streaming?episodeId=the-impossible-heir-2024-episode-1&mediaId=drama-detail/royal-loader&server=asianload
